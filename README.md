# Peluquería App 

Aplicación web para la gestión básica de una peluquería.  
Permite administrar **servicios** (corte, balayage, tintura, etc.), **clientes** y **turnos**, utilizando un frontend simple y un backend con persistencia real en **SQLite**.

El proyecto está pensado como un **sistema funcional de práctica / MVP**, con una arquitectura clara frontend + backend.

---

## Funcionalidades

- Gestión de servicios (corte, balayage, tintura, etc.)
- Gestión de clientes
- Gestión de turnos
- CRUD completo (Crear, Leer, Actualizar, Eliminar)
- Persistencia de datos con SQLite
- API REST simple

---

## Tecnologías utilizadas

### Frontend
- HTML5
- CSS3
- JavaScript 

### Backend
- Node.js
- Express.js

### Base de datos
- SQLite
